1.移除链表元素

删除链表中等于给定值 val 的所有节点

示例 1：
输入：head = [1,2,6,3,4,5,6], val = 6
输出：[1,2,3,4,5]

示例 2：
输入：head = [], val = 1
输出：[]

示例 3：
输入：head = [7,7,7,7], val = 7
输出：[]

思路：使用虚拟头结点，先让虚拟头结点指向head，设置当前节点 ，如果当前节点不为空，就进行判断，如果当前节点的值和val相等，就让虚拟节点的next，指向当前节点的next，如果不等，那就让虚拟节点=当前节点，当前节点=当前节点.next

最后返回虚拟节点的next

如上是思路，实际执行的时候，虚拟节点需要另找节点替代，否则最后返回时找不到头结点

2.设计链表

在链表类中实现这些功能：

  • get(index)：获取链表中第 index 个节点的值。如果索引无效，则返回-1。
  • addAtHead(val)：在链表的第一个元素之前添加一个值为 val 的节点。插入后，新节点将成为链表的第一个节点。
  • addAtTail(val)：将值为 val 的节点追加到链表的最后一个元素。
  • addAtIndex(index,val)：在链表中的第 index 个节点之前添加值为 val 的节点。如果 index 等于链表的长度，则该节点将附加到链表的末尾。如果 index 大于链表长度，则不会插入节点。如果index小于0，则在头部插入节点。
  • deleteAtIndex(index)：如果索引 index 有效，则删除链表中的第 index 个节点。

思路：

先初始化链表 给定size和head

get：如果给的index小于0或者大于size，返回-1，新建节点指向头，循环遍历index次，每次让当前节点=当前节点.next，当退出循环，说明找到了，返回当前节点.val

addAtIndex：如果大于size，就返回，如果小于0，就让index等于0，数据符合条件后，先让size+1，然后for循环找到要插入的位置，席间toAdd节点，让toAdd.next指向pred.next，prev.next指向toAdd，这里先后顺序也很重要，先指后置节点，不然toAdd找不到之后的方向

deleteAtIndex，找到要删除的index节点，让当前节点.next = 当前节点.next.next即可

剩下两个就是对addAtIndex操作
